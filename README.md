# python-relatorio-vendas

projeto em python para automatização de relatório de vendas utilizando análise de dados e envio automatizado de e-mails.

## Funcionalidades

- Leitura de planilha CSV.
- Cálculo do faturamento total.
- Contagem da quantidade de produtos vendidos.
- Geração automática do relatório.
- Envio do relatório por e-mail.

## Tecnologias utilizadas

- Python 3
- Pandas
- SMTP (envio de e-mail)

## Objetivo

Projeto desenvolvido para praticar automação de tarefas com Python, análise de dados e envio automático de e-mails.
